# eco-scan-challenge

## frontend
- reactjs 
- css 
- react router dom
- react hot toast
## backend 
- mongodb
- expressjs
- groq api (llama vision:11B model)
- nodejs
- cloudinary

## flow of application 
user uploads the data --upload function--> groq api gets called ----> calculates carbon_saved & reward_points ----> sends data to mongodb through cloudinary
